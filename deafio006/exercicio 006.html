Tags para o HTML

Em HTML5 podemos utilizar tags para
atingir objetivos especificos dentro de seus códigos, deixe o cursor
em cima de uma tag para saber sua função.

  • Parágrafo
  • Primeiro título
  • Abreviações
  • Listas Ordenadas
  • Listas Não Ordenadas
  • Negrito
  • Itálico
  • Imagem
  • Inserir
  • Texto desconsiderado
